Ruby on Rails(윈도우즈) 설치


  • 1. 루비(Ruby) 다운로드

    Railsinstaller.org에서 맞는 윈도우즈 옵션을 다운로드하세요.

  • 2. Installer Wizard 열기

    파일 다운로드가 끝나면 파일을 더블클릭해서 Installer Wizard를 열고 차례대로 'Next' 버튼을 누르고 설치합니다.

  • 3. Git Bash 열기

    '시작'메뉴에서 RailsInstaller폴더를 찾고, 폴더 안에서 'Git Bash'프로그램을 열어주세요.

  • 4. Rubygems 설치

    'Git Bash' 입력창에서 다음 코드를 입력한 후 ENTER를 누르세요.

    $ curl http://installrails.com/update_rubygems.rb | ruby

보다 자세한 설치 설명이 필요하면 installrails.com을 참고해주세.
