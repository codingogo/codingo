터미널(또는 iTerm)을 열고 아래 코드들을 입력하면서 설치를 따라해주세요.
(터미널 또는 iTerm을 Spotlight에서 열어주면됩니다.)


  • 1. 루비(Ruby) 설치 확인

    $ ruby -v
    # ruby 2.3.0

    ruby 2.3.0 (버젼은 상관 없음)이라고 나타나지 않으면 루비를 설치해야합니다. 루비 버젼이 나타나면 이미 설치가 되어 있는 것입니다.

  • 2. 래일즈(Rails) 설치 확인

    $ rails -v
    # Rails 4.2.6

    Rails 4.2.6 (버젼은 상관 없음)이라고 나타나지 않으면 래일즈를 설치해야합니다. 래일즈 버젼이 나타나면 이미 설치가 되어 있는 것입니다.

    • Ruby 또는 Rails가 설치되어 있지 않으면 아래 방법을 따라해주세요.

  • 3. 터미널에 Command Line 설치

    $ xcode-select --install

  • 4. Homebrew 설치

    $ ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

    $ brew -v
    # Homebrew 0.9.X

  • 5. RVM(Ruby Version Manager) 설치

    $ \curl -L https://get.rvm.io | bash -s stable

    터미널을 닫고 다시 열어서 다음 코드를 입력해서 루비를 설치합니다.

    $ rvm | head -n 1
    $ rvm use ruby --install --default

  • 6. 래일즈(Rails) 설치

    $ gem install rails --no-ri --no-rdoc

  • 7. 확인

    $ ruby -v
    # ruby 2.3.0

    $ rails -v
    # Rails 4.2.6

보다 자세한 설치 설명이 필요하거나 OS X 버젼이 10.9미만이면 installrails.com을 참고해주세요.


Ruby on Rails(윈도우즈) 설치


  • 1. 루비(Ruby) 다운로드

    Railsinstaller.org에서 맞는 윈도우즈 옵션을 다운로드하세요.

  • 2. Installer Wizard 열기

    파일 다운로드가 끝나면 파일을 더블클릭해서 Installer Wizard를 열고 차례대로 'Next' 버튼을 누르고 설치합니다.

  • 3. Git Bash 열기

    '시작'메뉴에서 RailsInstaller폴더를 찾고, 폴더 안에서 'Git Bash'프로그램을 열어주세요.

  • 4. Rubygems 설치

    'Git Bash' 입력창에서 다음 코드를 입력한 후 ENTER를 누르세요.

    $ curl http://installrails.com/update_rubygems.rb | ruby

보다 자세한 설치 설명이 필요하면 installrails.com을 참고해주세.
